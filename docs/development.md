# Development


## Dependencies

```bash
make deps
```

## Build

```bash
make build
bin/swage version
```


## Run

### on Local Machine

```bash
make run
```

> output >> swage.xlsx

### on Docker

```bash
# Linux, Darwin
make docker

# Windows
aio/scripts/docker.ps1

```

> output >> examples/testdata/docker-swage.xlsx
